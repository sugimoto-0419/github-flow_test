# github-flow_test

GitHub flowの練習用リポジトリ

## 手順

1. ブランチを作る
1. ファイルを編集
1. 不具合がなければコミット、プッシュする
1. プルリクエストを送る
1. レビューしてもらう
1. 何にもなければマージする
1. ブランチを削除する
1. ローカルにプルする

## おすすめ VScode 拡張機能
